### `<RegisterGsapPlugins>`

Registers custom **GSAP** plugins, if you need them for a specific use case.

#### Usage

```tsx
import { CSSRulePlugin } from "gsap/dist/CSSRulePlugin";
import { TextPlugin } from "gsap/dist/TextPlugin";

//...

useEffect(() => {
  RegisterGsapPlugins({
    plugins: [TextPlugin, CSSRulePlugin],
  });
//...
```
